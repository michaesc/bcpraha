Why store secrets?

  • Maintain order
  • Defend value
  • Assure privacy

How to store secrets?

  • In human memory
  • On paper notes
  • In computers
  • In metal safes

How hardware helps?

  • Provides storage
  • Interfaces with users
  • Processes block data
  • Communicates upstream

Special about blockchains?

  • Structured
  • Distributed
  • Transactional

How does a traditional hardware wallet work?

  • Isolated devices store a secret
  • Device listens for commands
  • Host sends transaction to sign
  • Device requests user authority
  • Device signs, sends signature
